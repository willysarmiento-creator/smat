# SMAT - Sistema de Monitoreo de Alerta Temprana

Este repositorio contiene la API RESTful para el sistema SMAT, desarrollada con FastAPI y SQLAlchemy. Cumple con los requisitos de la Evaluación Continua 1 (EV1).

## Instrucciones de Ejecución

1. Activar el entorno virtual.
2. Iniciar el servidor desde la raíz del proyecto:
   ```bash
   uvicorn app.main:app --reload
